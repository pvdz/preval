> Normalize > Expressions > Statement > Let > Auto ident call computed s-seq complex
>
> Normalization of all kinds of expressions should work the same no matter where they are

## Input

`````js filename=intro
let b = { $ };

let a = { a: 999, b: 1000 };
let xyz = (1, 2, b)[$("$")](1);
$(xyz);
$(a);
`````
